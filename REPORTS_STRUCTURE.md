# Advanced Attendance - Reports Structure

## Report Categories

### 1. **Attendance Reports**
- Daily Attendance Summary
- Monthly Attendance Report
- Employee Attendance Details
- Shift-wise Attendance Report

### 2. **Geofence & Location Reports**
- Geofence Violation Report
- Outside Geofence Check-ins
- Work Site Attendance Report
- GPS Accuracy Report

### 3. **Anomaly & Fraud Detection Reports**
- Device Fingerprint Anomaly Report
- Suspicious Activity Report
- Multiple Device Check-ins
- Anomaly Trend Analysis

### 4. **Time & Productivity Reports**
- Late Entry Report
- Early Exit Report
- Working Hours Summary
- Overtime Report

### 5. **Processing & Audit Reports**
- Attendance Processing Log
- Mobile Check-in Report
- Biometric Sync Report
- System Audit Trail

### 6. **Tour & Assignment Reports**
- Active Tour Plans
- Tour Plan Compliance
- Work Site Assignment Report

## Report Types

- **Script Report**: Python-based, complex queries
- **Query Report**: SQL-based, fast performance
- **Report Builder**: User-configurable

## Report Features

- Filters: Date range, employee, department, work site
- Export: Excel, PDF, CSV
- Scheduling: Email delivery
- Charts: Visual representations
- Drill-down: Detailed views
